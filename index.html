Help & Tips

Quick guidance for navigating TrailTrack and managing your routes.

Creating Routes

Use Create New Route to start drawing on the map or enable GPS tracking to record your path automatically.

Managing Routes

In the routes sidebar, view details, export GPX files, or remove routes using the action buttons next to each saved route.

Offline Use

Navigate to your desired area and select Download area for offline to cache map tiles before you lose connection.

Need Support?

Check the README for setup instructions or share feedback with GitHub Issues to help improve TrailTrack.
